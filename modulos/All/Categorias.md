# Módulo: Categorías

**Aplica a:** Todas las verticales (Dropshipping, Ecommerce, Gastronomía, Servicios)

## Descripción General
Este módulo permite la gestión de categorías para organizar los elementos principales del negocio (productos, platos, servicios, etc.).

## Funcionalidades Principales
- Creación, edición y eliminación de categorías.
- Asignación de imagen representativa.
- Ordenamiento y jerarquía.
