# Documentación de Módulos

Esta carpeta contiene la documentación técnica y funcional de los módulos del sistema Linkiu.bio.

## Estructura

- **All/**: Módulos transversales que aplican a todas o la mayoría de las verticales (ej. Categorías, Dashboard, Configuración).
- **Dropshipping/**: Módulos específicos para la vertical de Dropshipping (ej. Product Drop, Importación).
- **Ecommerce/**: Módulos específicos para Ecommerce avanzado.
- **Gastronomy/**: Módulos para restaurantes (ej. Mesas, Cocina, Comandas).
- **Services/**: Módulos para empresas de servicios (ej. Agenda, Citas, Equipo).

## Instrucciones
Para cada nuevo desarrollo o corrección, busca el archivo `.md` correspondiente al módulo o crea uno nuevo si no existe.
Usa estos archivos para:
1.  Documentar requerimientos.
2.  Listar tareas pendientes.
3.  Registrar decisiones técnicas.
